Use WebKit Scrollbar easy, browsers such as Chrome, Safari and Opera, supports the non-standard ::-webkit-scrollbar.

/* width */
::-webkit-scrollbar {
	width: 10px;
}

/* Track */
::-webkit-scrollbar-track {
	background: #f1f1f1;
}

/* Handle */
::-webkit-scrollbar-thumb {
	background: #888;
}

/* Handle on hover */
::-webkit-scrollbar-thumb:hover {
	background: #555;
}

The ::-webkit-scrollbar CSS pseudo-element affects the style of the scrollbar of an element.
